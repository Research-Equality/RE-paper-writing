# Submission Attachments

Many reporting standards are only fully satisfied when the manuscript is paired with extra artifacts.

## Common Attachments

- Reporting checklist PDF or journal form
- Participant or study-selection flow diagram
- Search strategy appendix for reviews
- Baseline characteristics table
- Protocol or registration link
- Data-availability statement
- Code or repository link
- Ethics approval statement
- Supplementary methods or extended results

## Practical Rules

- If a checklist item is answered in the supplement, point to the exact supplement section.
- Keep flow diagrams and baseline tables synchronized with the final manuscript numbers.
- Do not defer critical checklist evidence to "available upon request" unless the venue explicitly permits it.
- If data cannot be shared, explain the constraint and what metadata or derived outputs remain accessible.

## Before Submission

- Re-read the target venue instructions to see whether the checklist must be uploaded separately.
- Verify that all identifiers are stable: trial registration, repository DOI, accession number, supplement labels.
- Make sure every attached artifact uses the same terminology, sample counts, and arm names as the manuscript.
