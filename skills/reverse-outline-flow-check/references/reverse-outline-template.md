# Reverse Outline Template

Use this template for a section or subsection that feels unclear.

```text
Section goal / local thesis:

Paragraph 1
- Topic sentence:
- Main support or explanation:
- Role:
- Maps to section goal? yes / no / partial

Paragraph 2
- Topic sentence:
- Main support or explanation:
- Role:
- Maps to section goal? yes / no / partial
```

## How to Use It

- Start with the section goal, not the original paragraph order.
- Reduce each paragraph to one topic sentence and one support summary.
- If a paragraph cannot be summarized cleanly, it likely has multiple messages.
- If a paragraph does not map back to the section goal, revise or remove it.

## Role Labels

Useful role labels:

- opening
- challenge
- motivation
- method
- mechanism
- evidence
- limitation
- takeaway
