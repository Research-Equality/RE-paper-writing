# Claim Map Template

Use this template for the paper's headline claims.

```text
Claim:
Location:
Claim type:
Evidence anchor:
Support status:
Required action:
```

## Claim Types

- novelty
- performance
- robustness
- generalization
- efficiency
- practical value
- mechanistic explanation

## Support Status Labels

- `supported`: direct evidence is visible and aligned
- `partially-supported`: evidence exists but does not fully justify the wording
- `needs-evidence`: wording implies support that the paper does not show
- `not-testable-from-paper`: claim is interpretive or speculative and should be weakened or relocated

## Good Repair Actions

- weaken the scope
- add an explicit evidence anchor
- move the statement to Discussion
- delete the claim
- schedule a new experiment
