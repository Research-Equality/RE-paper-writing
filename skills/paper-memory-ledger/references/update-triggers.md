# Update Triggers

## Add to the Ledger When

- the target venue or submission strategy changes
- a major experiment finishes and its conclusion is stable
- the draft path or canonical bibliography path changes
- a reviewer requirement becomes binding for the next revision
- the team settles on terminology, figure order, or section structure that should persist

## Do Not Add When

- the idea is still exploratory
- the result has not been verified
- the decision might change in the next few minutes
- the note is just a transient task better stored in a TODO list

## Good Entry

`2026-04-01: switched target venue from workshop to NeurIPS-style short paper; appendix now carries extended ablations.`

## Bad Entry

`Maybe we should try a broader angle later if the current story feels weak.`
