# Polish Checklist

Run this before `submission-qa-gate`.

- Bullet lists are rare and intentional, not the default structure
- Adjacent sections do not repeat the same formatting pattern mechanically
- Figures are sized reasonably and float placement is not overly rigid
- `\newpage` appears only at real structural transitions
- Global list spacing is compact enough for a long manuscript
- Raw text-mode symbols such as `<` and `>` are not silently breaking the PDF
- Text after display equations does not start with accidental indentation
- The manuscript reads like a paper, not like notes or slide bullets
