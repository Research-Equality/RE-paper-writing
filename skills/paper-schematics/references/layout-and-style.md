# Layout and Style

## Layout Rules

- Use a clear reading direction: left-to-right or top-to-bottom.
- Group related elements with spacing before adding borders.
- Keep the number of visual primitives small enough that the figure can be scanned quickly.
- Reserve the strongest color for the main contribution or output, not for decoration.

## Labeling Rules

- Reuse manuscript terminology exactly.
- Prefer short noun phrases over sentence fragments.
- Keep in-figure text minimal and large enough to survive paper-scale rendering.

## Accessibility Rules

- Use colorblind-safe palettes.
- Ensure the figure still works in grayscale if printed.
- Differentiate groups with shape, line style, or position rather than color alone.

## Consistency Rules

- Match the color semantics and naming used in numeric figures.
- Keep panel labels, line widths, arrowheads, and fonts consistent across the paper.
- If the figure is conceptual, state that clearly in the caption so it is not mistaken for measured data.
