# Experiment Coverage Template

Map each contribution to the experiment that must validate it.

```text
Contribution:
Validation experiment:
Main evidence anchor:
Baseline requirement:
Ablation requirement:
Hard-setting / failure-mode requirement:
Status:
```

## Coverage Questions

- Which claimed benefit is the experiment supposed to validate?
- Is the supporting evidence direct or only suggestive?
- Are strong baselines included under fair settings?
- Is there an ablation for the mechanism being claimed?
- Is there any harder or more realistic setting that the paper should cover?
